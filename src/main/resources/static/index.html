Mod Privacy Notice

This mod uses a telemetry system to send anonymous information that helps us improve development and prioritize updates based on the most used versions. This information comes from other mods created by the same developer to provide a unified view of the usage of these mods. Here's how the collected data is handled:

1. Data Collected

  • Mod ID
  • Mod ID version
  • Minecraft version used
  • Loader type (Fabric, Forge, etc.)
  • Information about other mods authored by the same developer

2. Purpose of Data Collection

This data is collected solely for the purpose of:

  • Identifying the most widely used versions of a mod
  • Providing insights into the usage of other mods authored by the same developer
  • Prioritizing updates and improvements based on the gathered statistics

3. Data Anonymity

All collected data is completely anonymous. We do not use unique identifiers or any information that could identify users.

4. Data Use and Access

  • The collected data is not shared with third parties under any circumstances.
  • Access to the collected information is restricted and protected.

5. Disabling Telemetry

If you do not wish to participate in telemetry, you can easily disable it by editing the telemetry.json file located in the config directory and then in the folder with the mod id

6. Data Retention

Currently, the collected data is not deleted. This policy may change in the future to include more advanced data retention practices.

7. Transparency

This mod is open source, and you can review how telemetry works in the project repository: anvians-lib. You can also contribute to the development or discuss improvements related to data management.
